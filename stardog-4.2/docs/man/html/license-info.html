stardog-admin license info Manual Page

NAME


stardog-admin license info —Prints information about a Stardog license

SYNOPSIS


stardog-admin license info [--] <license file>

OPTIONS


--
This option can be used to separate command-line options from the list of argument, (useful when arguments might be mistaken for command-line options
<license file>
The license file to check

EXAMPLES


Print information about a license

    $ stardog-admin license info /path/to/my/stardog-license-key.bin
